# 命令
```
sudo npm install hexo-cli -g // 安装hexo的脚手架工具
hexo init blog // 初始化博客
cd blog // 返回博客目录
npm install // 安装依赖
hexo server // 启动博客

hexo server
// 简写
hexo s

hexo generate
// 简写
hexo g

hexo deploy
// 简写
hexo d

hexo new
// 简写
hexo n

hexo clean
hexo g
hexo d

// 打开ssh文件
open ~/.ssh

```
